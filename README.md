# Bus Tracking System (SyntexSparks)

A web-based **Bus Tracking System** that allows users to view live bus locations, routes, schedules, and enables drivers to update real-time location data. This project is designed to improve transparency, efficiency, and convenience in public or institutional transportation systems.

---

##  Features

###  User Module
- View live bus location on the map
- Select buses and routes
- Access bus schedules (PDF)
- Track buses in real time
- User-friendly interface

### 🚍 Driver Module
- Driver login interface
- Update live bus location
- Real-time data sync with Firebase

### Tracking & Map
- Live tracking using map interface
- Real-time location updates
- Route visualization

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Database:** Firebase  
- **Maps & Tracking:** Google Maps API (or Map-based JS integration)  
- **Hosting:** Firebase / GitHub Pages (optional)
